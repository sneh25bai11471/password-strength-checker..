Password Strength Checker

This project is a simple program that checks how strong a password is.

Problem:
Many people use weak passwords without realizing the risk. This can make their accounts easy to hack or misuse.

Solution:
In this project, I created a Python program that checks the strength of a password based on its length and the type of characters used, such as numbers, uppercase letters, and special symbols.

Tools Used:
Python (Google Colab)

Output:
The program tells whether the password is Weak, Medium, or Strong.
